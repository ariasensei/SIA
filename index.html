This course introduces students to the principles of System Integration and Software Architecture within the field of Game Development. Students will explore how various game systems—including graphics, physics, artificial intelligence, audio, networking, databases, user interfaces, cloud services, and external APIs—work together to create modern interactive games. Throughout the semester, students will learn integration models, software architecture patterns, middleware, event-driven systems, client-server communication, and deployment strategies used in professional game development. Laboratory activities emphasize hands-on implementation using industry-standard and open-source tools while encouraging collaboration, debugging, and architectural thinking. By the end of the course, students will be able to design, integrate, test, and evaluate complete game systems that follow modern software engineering and game development practices.
